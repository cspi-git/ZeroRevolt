# maskMention
Mask user mention.

## Installation
NpmJS:
```
npm i request-async
```

## Usage
```
node index.js <token> <channelID> <userID> <message>
```

- token - The Revolt account session token to use.
- channelID - The target channel(ID).
- userID - The target user(ID).
- message - The message to mask in the mention.
